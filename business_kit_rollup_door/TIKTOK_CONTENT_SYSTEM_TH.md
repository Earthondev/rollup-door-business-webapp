# TikTok Content System (งบต่ำ)

## Positioning
- แบรนด์: ทีมช่างมืออาชีพที่อธิบายงานตรงไปตรงมา
- ลูกค้าหลัก: เจ้าของร้าน/โกดังที่ต้องการงานเร็วและจบ

## เสาหลักคอนเทนต์ (5 คลิป/สัปดาห์)
1. ก่อน-หลังหน้างานจริง
2. ปัญหายอดฮิต + วิธีแก้
3. ความรู้เลือกสเปกให้เหมาะกับการใช้งาน
4. เบื้องหลังมาตรฐานความปลอดภัยทีมงาน
5. รีวิวลูกค้า/FAQ

## โครงคลิปมาตรฐาน 25-45 วินาที
- Hook 3 วินาทีแรก: ปัญหาหรือผลลัพธ์ชัด
- เนื้อหา: จุดเสีย -> วิธีแก้ -> ผลลัพธ์
- CTA เดียว: "ทัก LINE ส่งรูปหน้างาน + ขนาดคร่าวๆ"

## ตัวอย่าง Hook
- "ประตูโกดังฝืดทุกเช้า แก้ตรงไหนก่อน?"
- "งานนี้ลูกค้าคิดว่าจะต้องรื้อใหม่ทั้งหมด แต่เราแก้จบใน..."
- "3 จุดที่ทำให้ประตูเหล็กม้วนพังเร็วโดยไม่รู้ตัว"

## KPI คอนเทนต์รายสัปดาห์
- จำนวนคลิปที่ลง >= 5
- จำนวนอินบ็อกซ์จาก TikTok
- จำนวนลีดเข้าฟอร์ม LINE
- Conversion TikTok -> นัดสำรวจ

## Rule การผลิตคอนเทนต์
- ถ่ายหน้างานจริงเท่านั้น (ไม่ทำคอนเทนต์ลอย)
- ภาษาง่าย ไม่ขายเกินจริง
- ปิดท้ายด้วยวิธีติดต่อที่สั้นและซ้ำเดิมทุกคลิป
